### Ease / Lerp
```
let endValue = 0;

void draw() {
newValue = mouseX
endValue = lerp(endValue, newValue, 0.5);
}
```

### map and scaling range
```
// Distance example
distance = dist(eyeR.x, eyeR.y, eyeL.x, eyeL.y)
// Remapped value
remappedDist = map(distance, 0, 200, 20, 0)
```
